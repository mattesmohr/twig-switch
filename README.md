# Twig Switch

The package provides a switch tag for Twig.

```php
{% switch foo %}
    {% case 'bar' %}
    {% default %}
{% endswitch %}
```

## Questions?

### What is a switch tag?

A switch tag evaluates the expression and executes the matching case.

### What else?

The default clause handles the cases not explicitly listed.
